## 入门操作

安装依赖关系：

``` bash
$ git clone https://github.com/boy12371/kuangjia.org.git
$ cd kuangjia.org
$ npm install
```

生成：

``` bash
$ hexo generate
```

运行服务器：

``` bash
$ hexo server
```
